Below are a list of datasets broadly related to culture and the humanities, which might be useful for those interested in cultural analytics and the digital humanities.

Film üé¨¬∂

  • Film dialogue broken down by character gender and age (Download)

    • More info about this dataset

      Original Source: Hannah Anderson, Matt Daniels

      The Film dialogue broken down by character gender and age CSV file (.csv) is a consolidated and slightly modified version of data shared by Hannah Anderson and Matt Daniels for their Pudding article, ‚ÄúFilm Dialogue from 2,000 screenplays, Broken Down by Gender and Age‚Äù. The original datasets can be found on GitHub here.

      For 2,000 films from 1925 to 2015, the dataset includes information about characters‚Äô names, genders, ages, and how many words they spoke in each film (both by number of words and by overall proportion), as well as the release year of each film and how much money the film grossed.

      For more information about how Anderson and Daniels compiled the data and determined character gender and age, see FAQ for the ‚ÄúFilm Dialogue, By Gender‚Äù Project

Literature üìö¬∂

  • Harry Potter series, books 1-7, randomly shuffled (Download)

    • More info about this dataset

      Original Source: David Mimno

      The *Harry Potter* series text file (.txt) includes all seven books from J.K. Rowling‚Äôs Harry Potter series, with words randomly shuffled for each page. Each line of the file is tab-separated and contains information about the individual page number (e.g. hp1-15), the book number (e.g. hp1), and the randomly shuffled words for that page.

      This text file was originally created and shared by David Mimno. The words have been shuffled at the page level to ensure non-infringing fair use of copyrighted material.

  • txtLAB‚Äôs multilingual novels (Link)

    • More info about this dataset

      Original Source: Andrew Piper, McGill txtLAB

      The txtLAB‚Äôs multilingual novels link takes you to a repository where you can download a directory of 150 English-language novels, 150 German-language novels, and 150 French-language novels, which span from 1771 to 1932. Authors featured include Goethe, Franz Kafka, Hermann Melville, Mary Shelley, Kate Chopin, Virginia Woolf, Victor Hugo, Alexandre Dumas, and many more. These text files were compiled and shared by Andrew Piper and the txtLab.

  • Modernist journal data (1890s-1920s) (Link)

    • More info about this dataset

      Original Source: The Modernist Journals Project

      The Modernist journal data link takes you to a repository where you can download publication metadata for 14 modernist journals from the 1890s to the 1920s ‚Äî such as Poetry Magazine, The Little Review, and The Crisis. The Modernist Journals Project, which has digitized these journals, provides CSV and tab-separated text files that contain information for every contributor and every work published in the journals.

  • Seattle Public Library check-out data (2005-present) (Link)

    • More info about this dataset

      Original Source: City of Seattle

      The Seattle Public Library check-out data link takes you to a database that contains circulation data about the Seattle Public Library system from 2005 until the present. You can filter this database or search for keywords (e.g., ‚ÄúJames Baldwin‚Äù) and export a file of the filtered data by clicking ‚ÄúExport‚Äù and your desired file type.

Politics üó≥Ô∏è & History üìú¬∂

  • The New York Times obituaries (1852-2000) (Download)

    • More info about this dataset

      Original Source: Matthew Lavin

      The *New York Times* obituaries zip file (.zip) of text files (.txt) contains 378 New York Times obituaries (1852-2000) based on those collected by Matt Lavin for his Programming Historian tutorial, Analyzing Documents with TF-IDF.

      I re-scraped the 366 obituaries included in Lavin‚Äôs tutorial so that the obituary subject‚Äôs name and death year is included in each text file name. And I added 12 more ‚ÄúOverlooked‚Äù obituaries ‚Äî belated obituaries of remarkable women and minorities who did not receive a NYT obituary at the time of their death. Obituary subjects include academics, military generals, artists, athletes, activists, politicians, and businesspeople ‚Äî such as Ada Lovelace, Ulysses Grant, Marilyn Monroe, Virginia Woolf, Jackie Robinson, Marsha P. Johnson, Cesar Chavez, John F. Kennedy, Ray Kroc, and many more.

  • U.S. Inaugural Addresses (1789-2017) (Download)

    • More info about this dataset

      The U.S. Inaugural Addresses zip file (.zip) of text files (.txt) contains U.S. Inaugural Addresses ranging from President George Washington (1789) to President Donald Trump (2017). Each text file is titled with a number, the corresponding last name of the U.S. President, and the corresponding year of the Inaugural Address.

  • Nobel Prize winners (1901-2017) (Download)

    • More info about this dataset

      Original Source: The European Data Portal and the official Nobel Prize API

      The Nobel Prize winners CSV file (.CSV) contains information about 957 Nobel Prize winners from 1901 to 2017. This information includes the Nobel laureate‚Äôs name, birth and death date (if applicable), birth and death location (plus latitude and longitude coordinates for the locations), the year they won the Nobel Prize, the category of the Nobel Prize, and the ‚Äúmotivation‚Äù for the Nobel Prize.

      Nobel laureates include Marie Curie, Johannes Stark, Woodrow Wilson, Jane Addams, Rabindranath Tagore, John Steinbeck, Gabriel Garcia Marquez, Karl Ziegler, Toni Morrison, and many more.

  • Refugee arrivals to the U.S. (2005-2015) (Link)

    • More info about this dataset

      Original Source: Department of State‚Äôs Refugee Processing Center and Jeremy Singer-Vine

      The Refugee arrivals to the U.S. link takes you to a GitHub repository that contains data about refugee arrivals to the United States between 2005 and 2015. This data was originally compiled from the Department of State‚Äôs Refugee Processing Center by Jeremy Singer-Vine for his BuzzFeed article ‚ÄúWhere U.S. Refugees Come From ‚Äî And Go ‚Äî In Charts.‚Äù
      The ‚Äúrefugee-arrivals-by-destination‚Äù csv contains information about the number of refugees who arrived in each U.S. city and state, the year that they arrived, and the country from which they arrived. The ‚Äúrefugee-arrivals-by-religion‚Äù csv contains information about the number of refugees who arrived in the U.S., the year in which they arrived, and their religious affiliation.

  • Irish immigrants admitted to NYC‚Äôs Bellevue Almshouse (1840s) (Link)

    • More info about this dataset

      Original Source: Anelise Shrout, Digital Almshouse Project

      The Bellevue Almshouse link takes you to a Google spreadsheet that contains data about Irish-born immigrants who were admitted to the Bellevue Almshouse in the 1840s. The Bellevue Almshouse was part of New York City‚Äôs public health system, a place where poor, sick, homeless, and otherwise marginalized people were sent ‚Äî sometimes voluntarily and sometimes forcibly. This dataset was transcribed from the almshouse‚Äôs own admissions records by Anelise Shrout.
      For more information about this dataset, see The Almshouse Records

Social Media üï∏Ô∏è¬∂

  • Donald Trump‚Äôs tweets (2009-2020) (Download)

    • More info about this dataset

      Original Source: Trump Twitter Archive

      The Donald Trump's tweets CSV file (.csv) contains nearly 30,000 tweets from Donald Trump‚Äôs account from 2009 to March 2020. The information about each tweet includes the source, tweet text, date of tweet, as well as retweet and favorite counts. More updated data can be downloaded at Trump Twitter Archive.

  • ‚ÄúAm I The Asshole?‚Äù Reddit posts (Download)

    • More info about this dataset

      The Am I The Asshole?" Reddit posts CSV file (.csv) contains 2,932 Reddits posts from the subreddit ‚ÄúAm I the Asshole?‚Äù that have at least an upvote score of 2,000. The information in the dataset includes the date of the post, title, body text, url, upvote score, number of comments, and number of crossposts. This data was collected with PSAW, a wrapper for the Pushshift API.
