Open a Text File¬∂

If you want to read or write a text file with Python, it is necessary to first open the file. To open a file, you can use Python‚Äôs built-in open() function.

Let‚Äôs open the file ‚ÄúCharles-Dickens-excerpt.txt,‚Äù which is in our current directory.

open(FILEPATH HERE, mode='r', encoding='utf-8')

Inside the open() function parentheses, you insert the filepath to be opened in quotation marks. You should also insert a character encoding. This function returns what‚Äôs called a file object.

Read a Text File¬∂

A file object does not contain readable text yet. To read this file object as text, you need to use the .read() method.

open('Charles-Dickens-excerpt.txt', mode='r', encoding='utf-8').YOUR-CODE-HERE

What is \n?¬∂

print(open('Charles-Dickens-excerpt.txt', mode='r', encoding='utf-8').read())

Assign Text to a Variable¬∂

How would we assign this text to a variable?

Remix!¬∂

String Method

You can use the string method .replace() to replace text inside a string with different text. Replace the word ‚Äútimes‚Äù with a different word of your choice.

dickens.replace('times', 'tacos')

Let‚Äôs assign this remix to a new variable remixed_dickens.

remixed_dickens = dickens.replace('times', 'tacos')

Write a Text File¬∂

The default mode for the open() function is to read text files: mode = 'r'.

But you can use the open() function to write files, too. Simply set the mode to write: mode = 'w'

Let‚Äôs create a new text file with our remixed version of the opening line of A Tale of Two Cities.

open(NEW-FILE, mode='YOUR-CODE-HERE', encoding='utf-8').write(YOUR-CODE-HERE)

Double-click on your file in the file browser at the left and see if it worked!

Character Encodings¬∂

Why do we need to include encoding='utf-8' to open our text file?

UTF-8 is a character encoding known as Unicode. We need to specify a character encoding because ‚Äî gasp! ‚Äî computers don‚Äôt actually know what text is. Character encodings are systems that map characters to numbers.

You can check any characters‚Äô ‚Äúcode point,‚Äù or place in the Unicode universe, with the function ord()

ord("a")

ord("üí©")

ord("‡¶§")

ord("!")

Unicode is the most popular character encoding on the internet. It even includes emojis. Yet, as Aditya Mukerjee points out in his essay ‚ÄúI Can Text You A Pile of Poo, But I Can‚Äôt Write My Name‚Äù, Unicode still does not include characters that are essential to Bengali as well as to many other non-English languages.

Adding (UTF-8) Encoding¬∂

It‚Äôs always good practice to explicitly specify UTF-8 encoding when reading and writing files. Let‚Äôs open and read ‚Äúsample-character-encoding.txt‚Äù

sample_text_default = open('sample-character-encoding.txt', encoding='utf-8').read()
print(sample_text_default)

Look what happens if we read in the exact same text with a different encoding.

sample_text_iso = open('sample-character-encoding.txt', encoding='iso-8859-1').read()
print(sample_text_iso)

import re
from collections import Counter

def split_into_words(any_chunk_of_text):

Advanced¬∂

Read and Write Files Using a with Statement¬∂

There is another way that you can open files in Python, which is to use something called a with statement. You‚Äôll probably see this method if you look at code on the internet. The with statement helps ensure your files open and close properly. It‚Äôs more important for more advanced programming purposes, but it‚Äôs good to know about.

This method is very similar to the first method, except you put the word with before open(), and then you give the file object a nickname. We will call it read_file_object but you can actualy call it whatever you want (often people just use f but we want to be more descriptive!). This nickname as read_file_object is followed by a colon :. Then you read the file as normal.

with open('Charles-Dickens-excerpt.txt', encoding='utf-8') as read_file_object:
    text = read_file_object.read()

print(text)

You use the same structure for writing files but you might give the file object a different nickname write_file_object.

with open('text-file-made-by-with.txt', mode='w', encoding='utf-8') as write_file_object:
    write_file_object.write('I made this file with a with statement!')

Convert a File to UTF-8¬∂

One way to convert a file to UTF-8 is to open the file with the old encoding and then write a new version of the file with UTF-8 encoding.

Method 1

text_to_convert = open('../texts/literature/Lady-With-a-Dog_Chekov-KOI8R.txt', encoding='KOI8-R').read()

open('../texts/literature/Lady-With-a-Dog_Anton-Chekov-UTF8-Method1.txt', mode='w', encoding='UTF-8').write(text_to_convert)

Method 2

with open('../texts/literature/Lady-With-a-Dog_Chekov-KOI8R.txt', encoding='KOI8-R') as read_file_object:
    text_to_convert = read_file_object.read()
    with open('../texts/literature/Lady-With-a-Dog_Anton-Chekov-UTF8-Method2.txt', mode='w', encoding='UTF-8') as write_file_object:
        write_file_object.write(text_to_convert)
